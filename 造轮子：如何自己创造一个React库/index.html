从零开始构建一个react是很复杂的，但是借助现有的一些组件，快速实现react库的主要功能，则并不那么难。

我们构建的react库应当包含如下元素：

  • JSX
  • 函数组件
  • 类组件
  • 生命周期钩子函数

如无意外，该项目可以运行起来，我们并未使用官方的React库，而是自己创造了一个类似React的库，它叫做QndReact，可以参考 src/qnd-react.js 文件的代码。

关于 src/qnd-react.js

这个模块实现了 createElement 方法和 Component 基类。

前文有提到，编译 jsx 的时候，类似 <div></div> 会被编译为 React.createElement("div",... ，因此 createElement 不可或缺。

createElement 支持类组件和函数式组件，对两种组件使用的是不同的处理方式。

由于编译后的代码中，类组件也会变成函数，因此，增加了 isQndReactClassComponent 属性用于判断是否为类组件：

类组件中实现了 componentDidMount 生命周期钩子。

如果想实现更多生命周期函数，可查阅 snabbdom 文档。

关于 qnd-react-dom.js

QndReactDom 模块主要作用是将虚拟dom渲染到页面上。借助snabbdom库，propsModule 和 eventlistenersModule 提供了属性和事件的支持。

关于demo组件

src/counter.js 和 src/index.js 分别使用了类组件和函数式组件，使用方法官方React几乎完全一样。

到此为止，一个简易的、基于 snabbdom 实现的React库就完成了。但是目前 hooks 功能还是完全缺失的，还未考虑性能和兼容性问题。
